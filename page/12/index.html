Centos6.5搭建shdowsocks服务器

发表于 2016-03-05 | 更新于 2017-01-03 | 分类于 Other

yum update
yum install python-setuptools && easy_install pip
pip install shadowsocks
vi /etc/shadowsocks.json
[root@iZu1cf4uswmZ ~]# more /etc/shadowsocks.json
{
“server”:”47.88.50.29”,
“server_port”:8000,
“password”:”PA@@w0rd”,
“timeout”:300,
“method”:”rc4-md5”,
“fast_open”:true
}

######################################
代码中各字段的含义：

server：服务器 IP地址 (IPv4/IPv6)

server_port：服务器监听的端口，一般设为80，443等，注意不要设为使用中的端口

password：设置密码，自定义

timeout：超时时间（秒）

method：加密方法，可选择 “aes-256-cfb”, “rc4-md5”等等。推荐使用 “rc4-md5”

fast_open：true 或 false。如果你的服务器 Linux 内核在3.7+，可以开启 fast_open 以降低延迟。

workers：workers数量，默认为 1。

####################################################

开启
ssserver -c /etc/shadowsocks.json -d start
停止
ssserver -c /etc/shadowsocks.json -d stop

直接命令启动
/usr/bin/python /usr/bin/ssserver -p 8000 -k PA@@w0rd -m rc4-md5 -d start

Memcached-1.4.24启动脚本

发表于 2016-03-05 | 更新于 2017-01-03 | 分类于 Middleware

启动脚本在源码包目录下：
/root/backup/memcached-1.4.24/scripts/memcached.sysv
拷贝到init.d下面用：
cp /root/backup/memcached-1.4.24/scripts/memcached.sysv /etc/init.d/memcached
新建var下目录：
mkdir /var/run/memcached
需要修改安装路径：

改权限：chmod 755 memcached
加到开机服务：
chkconfig –add memcached
设置开机自启动：
chkconfig memcached on

Php-5.4.45启动脚本

发表于 2016-03-05 | 更新于 2017-01-03 | 分类于 Middleware

启动脚本在源码包目录下：
/root/backup/php-5.4.45/sapi/fpm/init.d.php-fpm
拷贝到init.d下面直接用：
cp /root/backup/php-5.4.45/sapi/fpm/init.d.php-fpm /etc/init.d/php-fpm
改权限：chmod 755 php- fpm
加到开机服务：
chkconfig –add php-fpm
设置开机自启动：
chkconfig php-fpm on
