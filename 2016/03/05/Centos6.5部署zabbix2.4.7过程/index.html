Centos6.5部署zabbix2.4.7过程

Posted on 2016-03-05 | In Zabbix

首先配置zabbix服务器能通外网
关闭iptables和selinux
安装zabbix存储库包
rpm -ivh http://repo.zabbix.com/zabbix/2.4/rhel/6/x86_64/zabbix-release-2.4-1.el6.noarch.rpm
安装zabbix
yum install zabbix-server-mysql zabbix-web-mysql
安装mysql
yum install mysql-server -y
安装ntp
yum install ntp -y
ntpdate time.nist.gov
设置开机启动
chkconfig ntpd on
chkconfig httpd on
chkconfig mysqld on
chkconfig zabbix-server on
启动服务
/etc/init.d/ntpd start
/etc/init.d/httpd start
/etc/init.d/mysqld start
/etc/init.d/zabbix-server start
创建zabbix数据库
mysql -uroot -p
create database zabbix character set utf8 collate utf8_bin;
设置mysql访问权限，只允许本机访问
grant all privileges on zabbix.* to zabbix@localhost identified by ‘zabbix’;
flush privileges;
quit；
导入数据表结构
cd /usr/share/doc/zabbix-server-mysql-2.4.7/create
mysql -uzabbix -pzabbix zabbix < schema.sql
mysql -uzabbix -pzabbix zabbix < images.sql
mysql -uzabbix -pzabbix zabbix < data.sql
编辑zabbix配置文件指向数据库
vi /etc/zabbix/zabbix_server.conf
DBHost=localhost
DBName=zabbix
DBUser=zabbix
DBPassword=zabbix

修改httpd下的zabbix配置文件
vi /etc/httpd/conf.d/zabbix.conf
php_value max_execution_time 300
php_value memory_limit 128M
php_value post_max_size 16M
php_value upload_max_filesize 2M
php_value max_input_time 300
php_value date.timezone Asia/Shanghai
service httpd restart

添加iptables规则
-A INPUT -p tcp -m tcp –dport 10050 -j ACCEPT
-A INPUT -p udp -m udp –dport 10050 -j ACCEPT
-A INPUT -p tcp -m tcp –dport 10051 -j ACCEPT
-A INPUT -p udp -m udp –dport 10051 -j ACCEPT

进入web安装向导
http://服务器IP/zabbix

完成

完成后登陆
登陆链接：http://192.168.10.222/zabbix/index.php
默认账号Admin，密码zabbix

修改中文语言

# zabbix
Mysql-5.7.10安装配置&主备
Zabbix添加php-fpm监控
