Centos7.2部署elk过程

Posted on 2016-03-05 | In ELK

安装java环境
yum install java-1.7.0-openjdk

安装lasticsearch
rpm -ivh elasticsearch-2.1.1.rpm

安装logstash
rpm -ivh logstash-2.1.1-1.noarch.rpm

安装kibana并配置supervisor
解压
tar zxvf kibana-4.3.1-linux-x64.tar.gz
拷贝到/etc/目录
cp -r kibana-4.3.1-linux-x64 /etc/kibana4.3.1

安装supervisor
yum install python-setuptools

easy_install supervisor
生成配置文件：

echo_supervisord_conf >> /etc/supervisord.conf
编辑配置文件vi /etc/supervisord.conf
最后添加：
[include]
files = /etc/supervisord.d/*.conf

添加kibana配置文件：
mkdir /etc/supervisord.d
vi kibana.conf
添加：
[program:kibana]
command=/etc/kibana4.3.1/bin/kibana
autorestart = true
redirect = true
stderr_logfile = /var/log/kibana/kibana.log
startsecs = 3

给日志文件建目录：
mkdir /var/log/kibana
touch /var/log/kibana/kibana.log
启动：

/usr/bin/python /usr/bin/supervisord -c /etc/supervisord.conf

防火墙打开5601端口
firewall-cmd –permanent –add-port=5601/tcp
firewall-cmd –reload
查看防火墙端口情况
firewall-cmd –list-all

配置elastcisearch
修改配置文件：
vim /etc/elasticsearch/elasticsearch.yml
修改如下几处

启动：
systemctl daemon-reload
systemctl enable elasticsearch.service
systemctl start elasticsearch
systemctl status elasticsearch

使用9200和9300端口，防火墙打开端口
firewall-cmd –permanent –add-port={9200/tcp,9300/tcp}
firewall-cmd –reload
查看防火墙端口情况
firewall-cmd –list-all

配置logstash
添加配置文件：
vim /etc/logstash/conf.d/nginxconf.json

防火墙打开5044端口
firewall-cmd –permanent –add-port=5044/tcp
firewall-cmd –reload
查看防火墙端口情况
firewall-cmd –list-all

启动：
systemctl start logstash
systemctl status logstash
chkconfig logstash on (开机启动设置特殊)
